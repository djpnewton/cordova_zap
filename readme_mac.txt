Requirements
------------

 - Xcode: install from the app store, needed for git, compilers etc
 - Android Studio: make sure the NDK component is installed and the SDK build tools
 - homebrew: https://brew.sh for installing wget etc..
 - wget: 'brew install wget'
 - cmake: 'brew install cmake'
 - nodejs: 'brew install nodejs'
 - cordova: 'npm install -g cordova'
 - java8: 'brew tap caskroom/versions; brew cask install java8'
 - gradle: 'brew install gradle'
